# Cloudspire AI

Bug reports, feature requests, feedback, and roadmap for Cloudspire AI — intelligent voice agents, chat agents, and SMS automation.

## How to Submit

1. Go to the [Issues tab](../../issues)
2. Click **New issue**
3. Choose **Bug Report** or **Feature Request**
4. Fill out the form and submit

## What Belongs Here

- **Bug reports** — Something isn't working correctly in Cloudspire AI
- **Feature requests** — Suggest improvements or new capabilities
- **Feedback** — General feedback on the AI platform
- **Roadmap items** — Track committed and planned features

## Scope

This tracker covers all Cloudspire AI features:

- AI voice agents (inbound call handling, Retell integration)
- AI chat agents (web chat widget, conversational AI)
- SMS/chat automation (AI-powered SMS responses)
- Agent configuration and management
- AI billing and usage tracking
- Knowledge bases and training
- Demo agents and templates

> For Cloudspire Voice issues (PBX, call routing, phone provisioning), use [cloudspire-voice](https://github.com/onCloud-Cloudspire/cloudspire-voice).

## Labels

| Label | Meaning |
|---|---|
| `bug` | Something isn't working correctly |
| `enhancement` | New feature or improvement request |
| `roadmap` | Committed to the product roadmap |
| `under-review` | Submitted but not yet committed to roadmap |
| `critical` | System down or data loss |
| `high` / `priority-high` | Major issue, no workaround |
| `medium` / `priority-medium` | Partial issue, workaround exists |
| `low` / `priority-low` | Minor or cosmetic |
| `docs` | Documentation |
| `resolved` | Fixed, pending verification |
| `duplicate` | Already reported |
| `wontfix` | Not planned |
